mapyTracks

O co jde

Použít hosting HTML stránek na GitHubu a api api4mapy k zobrazování tracklogů na mapových podkladech mapy.cz. Výsledná mapa by měla jít vložit do frame na libovolné stránce.

GitHub by měl přinést kromě možnosti webhostingu také verzování jednotlivých tras i samotné javascriptové aplikace.
